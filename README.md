# IMDB Sentiment Analysis

This project builds a sentiment analysis model on the IMDB movie review dataset.

## Dataset
IMDB Movie Review Dataset (25,000 training + 25,000 testing samples)

## Method
- Text representation using word embeddings
- Neural network implemented with TensorFlow / Keras
- Binary classification (positive vs negative)

## Model
Embedding Layer  
GlobalAveragePooling  
Dense Layer (sigmoid)

## Result
Achieved about 85% accuracy on the test dataset.

## Tech Stack
Python  
TensorFlow / Keras  
NumPy
